1. Classes And Id

Classes is defined as .class_name{} in <style> </style> tag.

And Id are defined as #id_name() in <style> </style> tag.

  • Code:


    • < div class="red underline bold" > Hello < /div >

  • In Style


    • < style >

    .red{ color: red;}
    .underline{ text-decoration: underline;)
    .bold{ font-weight: bold;)
    OR

    Use < strong > < /strong > for bold.

  • Output


    • Hello
